# VfHub – v1.0 (VirulentPred2)

## Overview

**VfHub** is a curated repository of **putative bacterial virulence factors** predicted using **VirulentPred2**, a machine‑learning tool for bacterial virulence protein prediction.

For each species, a **species‑specific** FASTA file of predicted VFs is provided.  
In addition, a **single, combined** FASTA file contains **all non‑virulent sequences** (negative controls) from every species analysed. This file is updated as new species are added.
